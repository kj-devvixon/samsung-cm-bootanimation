# CyanogenMod Bootanimation for Samsung A51
Magisk module replacing the Samsung/Carrier (Like Plus/Verizon and etc.) logo with the classic Cid.

##  Installation Guide

### Prerequisites
* A Samsung Galaxy A51 (or other Samsung device with similar screen ratio).
* Device must be **Rooted** with **Magisk**.

### Steps
1. Download the latest `cm_boot_mod.zip` from the [Releases](https://github.com/kj-devvixon/samsung-cm-bootanimation/releases) section.
2. Open the **Magisk App** on your phone.
3. Go to the **Modules** tab (puzzle icon at the bottom).
4. Tap **Install from storage**.
5. Select the downloaded `.zip` file.
6. Once the installation is finished, tap **Reboot**.

### ⚠️ Note
If you still see the carrier logo after reboot, it might be stored in a different partition (like `/product/media` or `/prism`). If that's the case, feel free to open an Issue in this repository!
